# 🍽️ Zomato Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing Zomato restaurant data to uncover meaningful insights about restaurant trends, ratings, votes, and customer preferences. Using Python and data visualization techniques, the dataset was explored to understand patterns in restaurant services and customer behavior.

The goal of this project is to perform **Exploratory Data Analysis (EDA)** and transform raw data into useful insights through visualizations.


## 📊 Objectives
- Clean and preprocess the Zomato dataset
- Analyze restaurant ratings and vote distributions
- Explore restaurant types and service trends
- Study the impact of online ordering on ratings
- Visualize insights using graphs and charts


## 🛠️ Technologies Used
- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Jupyter Notebook**


## 📂 Project Workflow
1. **Data Collection**
   - Imported Zomato dataset.

2. **Data Cleaning**
   - Handled missing values
   - Converted data types
   - Removed inconsistencies

3. **Exploratory Data Analysis (EDA)**
   - Analyzed restaurant types
   - Examined rating distributions
   - Studied votes and customer preferences
   - Explored online order trends

4. **Data Visualization**
   - Count plots
   - Histograms
   - Boxplots
   - Heatmaps


## 📈 Key Insights
- Certain restaurant types receive higher customer votes.
- Online ordering services influence restaurant popularity.
- Ratings distribution shows how customers evaluate restaurants.
- Dining cost trends vary across different restaurant categories.
